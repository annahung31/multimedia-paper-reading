## Key contribution

假如 SDV(Self-Driving Vehicle) 需要高解析度的 map 才能駕駛，那當 map 有誤差、或者沒有即時更新的時候，就很容易做出錯誤決策，發生危險。因此，本篇提出在沒有 HD maps 的狀況下，使用收到的 raw sensor data 和 high level 指令，得到可以解釋的 representation，並做出更安全的駕駛決定。
